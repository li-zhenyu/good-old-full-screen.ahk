Most of the time you want to switch a window to fullscreen mode just for hidding its decoration and the task bar. This works. 

Don't use those bullshit `F11` scripts any longer. Try my one.

## How to use it?

Install AutoHotKey version 2, then double click.

`Win`+`F` to toggle fullscreen or not.
